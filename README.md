## Installation

### Carthage

Add the following to your Cartfile:

```
github "Endoze/BaseTask"
```

Then add the framework as a linked framework.

### CocoaPods

Add the following to your Podfile:

```
use_frameworks!


pod 'BaseTask'
```

Then run `pod install`
